Version 1.4.4 (2024-??-??)
--------------------------

  * Updated the language server to the latest version.

  * The language server now supports inlay hints, which can show the types of
    variables and function parameters. Inlay hints must be manually enabled in
    the LSP settings ("Preferences: LSP Settings" from the command palette) via
    "show_inlay_hints": true   - they are disabled by default because there are
    some known bugs for inlay hints.
